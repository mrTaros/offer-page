# Offer Page Generator — PRD

## Назначение

Внутренний клиентский инструмент для HR-отдела MetaRatings. Автоматически форматирует текст оффера в фирменный шаблон компании с корректной типографикой, цветами, иконками и логотипом. Результат — превью на экране + скачивание PDF / DOCX.

## Стек

- HTML5 (single page)
- Tailwind CSS (CDN)
- Vanilla JavaScript
- Google Fonts (Oswald)
- html2pdf.js (CDN) — экспорт PDF
- docx.js (CDN) — экспорт DOCX

## Структура файлов

```
private/offer-page/
  index.html
  docs/PRD.md
  assets/
    logo-white.svg
    logo-color.svg
    icons/         -- SVG-иконки секций и преимуществ
  README.md
```

## Шаблон оффера (секции)

1. Хедер — "Welcome on board!" на фиолетовом фоне (#601DC0) + белый логотип
2. Приветствие — имя, вводный текст, роль
3. Условия взаимодействия — key:value пары
4. Ключевая цель роли — параграф
5. Метрики успеха роли — маркированный список
6. Зона ответственности — маркированный список
7. Наши преимущества — предзаполнен, редактируемый
8. Почему Metaratings — финальный параграф
9. Футер — срок действия + логотип

## Брендинг

- Заголовки: Oswald (Google Fonts)
- Тело: Helvetica Neue / Arial / sans-serif (системный)
- Акцент: #601DC0 (Purple Heart)
- Текст: #252525 (Mine Shaft)
- Серый: #5B5B5B (Scorpion)
- Фон: #FFFFFF

## Формат входного текста

Секции разделяются заголовками-маркерами (регистронезависимо):

```
Имя: ...
Роль: ...
Приветствие: ...
Условия взаимодействия: ...
Ключевая цель роли: ...
Метрики успеха роли: ...
Зона ответственности: ...
Почему Metaratings: ...
Оффер действителен до: ...
```

## Функциональные требования

- FR-1: Textarea + загрузка .txt файла
- FR-2: Парсинг текста по маркерам-заголовкам
- FR-3: Превью оффера по шаблону
- FR-4: Редактируемый блок «Наши преимущества»
- FR-5: Экспорт PDF (A4)
- FR-6: Экспорт DOCX
- FR-7: Валидация — подсветка недостающих секций

## Нефункциональные требования

- Полностью клиентское (без бэкенда)
- Работает локально (file://) и на статическом хостинге
- Chrome, Safari, Firefox (последние версии)
- Интерфейс на русском
