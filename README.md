# Offer Page Generator

Внутренний инструмент для генерации брендированных офферов MetaRatings.

## Запуск

Откройте `index.html` в браузере (Chrome, Safari, Firefox).

Серверная часть не требуется — всё работает на клиенте.

## Использование

1. Вставьте текст оффера в текстовое поле или загрузите файл (PDF, DOCX, TXT).
2. Проверьте блок «Сверка пунктов оффера» — он покажет, какие секции найдены, а каких не хватает.
3. При необходимости отредактируйте блок «Наши преимущества».
4. Нажмите **Сгенерировать оффер**.
5. Проверьте превью на экране.
6. Скачайте результат в PDF или DOCX.

## Поддерживаемые форматы загрузки

- `.txt` — читается как текст (UTF-8)
- `.pdf` — текст извлекается через PDF.js
- `.docx` / `.doc` — текст извлекается через Mammoth.js

## Триггеры парсера

Парсер ищет в тексте следующие секции. Двоеточие после заголовка необязательно.

### Обязательные

| Триггер | Как определяется |
|---|---|
| **Имя** | Из строки «Имя, привет» / «Имя привет» |
| **Роль** | Из фразы «в роли — ...» |
| **Условия взаимодействия** | Заголовок секции |
| — Формат | Подполе внутри «Условия» |
| — Вознаграждение | Подполе внутри «Условия» |
| — Местоположение | Подполе внутри «Условия» |
| — Режим вовлечённости | Подполе внутри «Условия» |
| — Координатор задач | Подполе внутри «Условия» |
| — Тестовый период | Подполе внутри «Условия» |
| **Ключевые цели роли** | Заголовок секции |
| **Зона ответственности** | Заголовок секции |
| **Наши преимущества** | Заголовок секции |
| **Действителен до** | Дата из любого места текста |

### Опциональные

| Триггер | Описание |
|---|---|
| **Метрики успеха роли** | Если есть — добавляется в оффер, если нет — пропускается |
| **Почему Metaratings** | Финальный мотивационный блок |

```

Благодарим тебя за интерес к нашей команде и проделанный путь в рамках
отбора. Мы рады предложить тебе сотрудничество в роли -

Условия взаимодействия:
Формат сотрудничества: Самозанятость
Вознаграждение: 140 000 рублей Net
Местоположение: Полностью удаленное сотрудничество
Режим вовлечённости: по согласованию с координатором задач
Координатор задач: Head of Back Office - Мария Вилистер
Тестовый период: 3 месяца

Ключевые цели роли:
Обеспечить оформление и сопровождение сотрудников
...

Зона ответственности:
Консультация по формату оформления (СЗ/ИП/ТК)
...

Наши преимущества:
Гибкий график и удалённая работа
...

Почему Metaratings — это для тебя?
Мы ценим вклад каждого...

Обращаем твое внимание, что оффер действителен до 08.03.2026
```

## Дизайн

Оффер генерируется в фирменном стиле MetaRatings:
- Фиолетовый фон (#601DC0) с белой карточкой контента
- Шрифт заголовков: Oswald, шрифт текста: Helvetica Neue
- PNG-иконки секций и логотип из `assets/`

```
private/offer-page/

  README.md
```
